# 🚀 MOLNIY KG Bot — Орнотуу нускамасы

## 1. Алдын ала керектүүлөр
- Node.js v18+ орнотулган болуш керек
- Telegram аккаунту

## 2. Орнотуу

```bash
# Папканы ач
cd molniy_bot

# Зависимостьтерди орнот
npm install
```

## 3. Токенди алуу
1. Telegramда @BotFather ботун ач
2. /newbot деп жаз
3. Ботуңа ат бер
4. BotFather сага токен берет — мисалы: `7123456789:AAHxxxxx...`

## 4. Токенди боттун файлына кой
`bot.js` файлын ач, биринчи сапты тап:
```javascript
const TOKEN = 'YOUR_BOT_TOKEN_HERE';
```
Ошол жерге өз токенүңдү коюп чыгар.

## 5. Фотолорду кой (каалоо боюнча)
`bot.js` ичиндеги `placeholder` сүрөт URL'дарын өзүңдүн сүрөт URLдарыңа алмаштыр.

## 6. Иштетүү

```bash
node bot.js
```

## 7. Иштеп жаткан белгиси
Консолдо `🚀 MOLNIY KG Bot запущен!` деп чыкса — бот иштеп жатат.
